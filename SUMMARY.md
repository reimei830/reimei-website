## 💡 ヒント

### ローカルで確認する方法
```bash
cd reimei-tech
npm install
npm run dev
```
→ http://localhost:3000 で確認

### 本番ビルドを確認
```bash
npm run build
```
→ `out/` フォルダに生成

### 更新をデプロイ
```bash
git add .
git commit -m "更新内容"
git push
```
→ Vercelが自動的に再デプロイ

## 🆘 サポート

### よくある質問

**Q: ビルドエラーが出る**
A: `npm install` を実行して、依存関係を再インストール

**Q: ドメインが反映されない**
A: DNS設定が正しいか確認。最大48時間かかる場合があります

**Q: 画像が表示されない**
A: `public/` フォルダに画像があるか、ファイル名が正しいか確認
